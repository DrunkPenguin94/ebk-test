Код с моделями в common/models .

Консольная версия:
	Код находится в console/controllers.

	Проверка алгоритма:
	yii api/index %n% %str%   
	где %n% - число (пример "5"), %str% - массив чисел в виде строки (пример "5, 5, 1, 7, 2, 3, 5")
	//yii api/index "5" "5, 5, 1, 7, 2, 3, 5"
	
	Создание пользователя 
	yii api/newuser %login% %pass%   Newuser
	где %login% - строка с логином (пример "test"), %pass% - строка с паролем (пример "11111")
	//yii api/newuser "test" "111111"
	
	Миграция
	yii migrate
	
Rest:
	Код находится в frontend/controllers.
	
	
	Проверка алгоритма:
	/index - сделал форму , с полями токен (который можно получить через консоль при создании юзера), числом и массивом чисел. 
	Аяксом идет запрос к ресту и возвращается результат.
